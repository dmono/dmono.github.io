Latest Project: Xorro P2P

Due to an interest in P2P networks and distributed systems, my team and I set out to build our own from scratch. The result is Xorro P2P, a BitTorrent-like peer-to-peer file sharing network. It includes a web UI that provides a simple interface for users to distribute and retrieve files in a network.

Visit the Xorro P2P project site to learn more
